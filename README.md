# HUL Financial Modeling & DCF Valuation

This project features an integrated three-statement financial model and Discounted Cash Flow (DCF) valuation for **Hindustan Unilever Ltd. (HUL)**. The model projects financial performance over a five-year horizon to derive an intrinsic value for the company under various macroeconomic scenarios.

## 📊 Key Features
*   **Integrated Three-Statement Model:** Interconnected Income Statement, Balance Sheet, and Cash Flow Statement projections.
*   **DCF Valuation:** Unlevered Free Cash Flow to Firm (FCFF) valuation model.
*   **Stress-Testing:** Scenario-based revenue growth modeling to quantify downside risk in adverse market conditions.
*   **Strategic Metrics:** Calculation of Weighted Average Cost of Capital (WACC) and terminal value assumptions for intrinsic pricing.

## 📁 File Structure
*   **HUL_Financial_Model.xlsx**: The primary financial model containing the historical data, assumptions, projections, and valuation outputs.

## 🛠️ Modeling Assumptions
The valuation is built upon the following core financial assumptions:
*   **Forecast Horizon:** 5 Years
*   **WACC (Discount Rate):** 10.5%
*   **Terminal Growth Rate:** 5.0%

## 📈 Methodology
1.  **Forecasting:** Projecting operational performance based on macroeconomic trends and historical financial data.
2.  **Valuation:** Constructing the FCFF model to determine intrinsic value based on discounted future cash flows.
3.  **Risk Analysis:** Stress-testing key revenue drivers to generate a range of investment-grade valuation outcomes.
